Проекты пользователя:
Входные:
+-Количесвто проектов пользователя + его максимальное количесвто

+проект:
	название
	дата создания
	включен или выключен
	количество ресурсов максимум и сколько использовано

возможности проекта:
	включение выключение
	копировать проект
	+клонировать проект
	+редактироват проект
	+удалить проект

+создание проекта:
	название проекта
	префикс (/api/v1)

Ресурсы внутри проекта:
Создание:
	название
	+-генератор данных для json
	создание иерархии объекта

Функционал:
	+генерирование количество объектов
	изменение сгенерированных данных
	изменение настроект ресурса
	удаление ресурса



генерация данных

будет принимать объект в котором будут находитсья названия поля, и его типы данных

генерация объекта названия поля, тип поля (тип поля + под тип) 1 строка id с uuid
